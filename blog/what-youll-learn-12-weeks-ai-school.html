Phase 1: Building the Foundation (Weeks 1–3)

Every great AI practitioner starts with strong fundamentals. The first three weeks of our program ensure you have a rock-solid grasp of the programming and mathematical concepts that underpin all of machine learning and artificial intelligence.

Week 1 — Python Foundations & Dev Environment

You'll start by setting up a professional development environment: VS Code, Git, virtual environments, and the command line. From there, you'll dive into Python — covering data types, control flow, functions, object-oriented programming, and file I/O. By the end of the week, you'll write clean, modular Python code and push your first project to GitHub.

Week 2 — Data Manipulation & Visualization

Week two introduces the data science toolkit: NumPy for numerical computing, Pandas for data manipulation, and Matplotlib and Seaborn for visualization. You'll work with real-world datasets — cleaning messy data, performing exploratory data analysis (EDA), and generating insights through charts and statistical summaries. Your first mini-project: an end-to-end EDA report on a public dataset of your choosing.

Week 3 — Math for Machine Learning

We don't believe in skipping the math. Week three covers the essential linear algebra, probability, statistics, and calculus concepts you'll need to truly understand what's happening under the hood of ML algorithms. You'll learn about vectors, matrices, gradients, probability distributions, and Bayes' theorem — all taught through applied Python exercises rather than abstract proofs.

Phase 2: Machine Learning & Deep Learning (Weeks 4–6)

With the foundation set, you'll spend the next three weeks going deep into classical machine learning and modern deep learning. This is where things get exciting.

Week 4 — Classical Machine Learning

Using scikit-learn, you'll implement supervised learning algorithms including linear regression, logistic regression, decision trees, random forests, and support vector machines. You'll also cover unsupervised methods like k-means clustering and PCA. Crucially, you'll learn the full ML workflow: train/test splits, cross-validation, hyperparameter tuning, and evaluation metrics like precision, recall, F1, and AUC-ROC.

Week 5 — Deep Learning with PyTorch

Welcome to neural networks. You'll learn the theory behind perceptrons, activation functions, backpropagation, and gradient descent — then implement it all in PyTorch. Starting with simple feedforward networks, you'll progress to convolutional neural networks (CNNs) for image classification. Your project: building an image classifier that achieves over 90% accuracy on a benchmark dataset.

Week 6 — Sequence Models & NLP Fundamentals

Week six shifts focus to sequential data. You'll study recurrent neural networks (RNNs), LSTMs, and the attention mechanism that revolutionized the field. On the NLP side, you'll cover tokenization, word embeddings, and text classification. The week's project involves building a sentiment analysis model using both classical and deep learning approaches, giving you a direct comparison of the two paradigms.

Phase 3: Large Language Models & Generative AI (Weeks 7–9)

This is where Austin AI School's curriculum truly differentiates itself. While many programs stop at deep learning, we go further — into the technologies dominating the current AI landscape.

Week 7 — Transformer Architecture & LLMs

You'll study the Transformer architecture in depth — self-attention, multi-head attention, positional encoding, and the encoder-decoder framework. Then you'll explore the landscape of large language models: GPT, LLaMA, Mistral, and Claude. You'll learn about pretraining vs. fine-tuning, prompt engineering best practices, and how to work with the OpenAI API and Hugging Face Transformers library to leverage state-of-the-art models in your own applications.

Week 8 — AI Agents & Tool Use

AI agents are the next frontier. This week you'll learn how to build autonomous AI systems that can reason, plan, and take actions using tools. Using LangChain and LangGraph, you'll construct agents that can browse the web, query databases, execute code, and interact with external APIs. You'll implement ReAct-style reasoning loops, manage agent memory, and build multi-step workflows. Your project: a research agent that can answer complex questions by autonomously gathering and synthesizing information from multiple sources.

Week 9 — Retrieval-Augmented Generation (RAG)

RAG has become the industry-standard approach for grounding LLMs in domain-specific knowledge. You'll build complete RAG pipelines using vector databases like Pinecone and ChromaDB, embedding models, chunking strategies, and retrieval techniques. You'll learn about hybrid search, re-ranking, and how to evaluate RAG system quality. The week culminates in building a knowledge-base chatbot that can answer questions about a large document corpus with citations — a project template directly applicable to enterprise use cases.

Phase 4: Production & Deployment (Weeks 10–11)

Building a model in a notebook is one thing. Shipping it to production is another. These two weeks bridge that critical gap.

Week 10 — MLOps & Model Deployment

You'll learn to package AI applications using FastAPI and Docker, deploy them to cloud platforms like AWS and Google Cloud, and set up CI/CD pipelines. Topics include model versioning with MLflow, monitoring model performance in production, handling API rate limiting, and cost optimization. You'll deploy your RAG chatbot from the previous week as a live, publicly accessible web service.

Week 11 — Fine-Tuning, Evaluation & Safety

In week eleven, you'll fine-tune open-source LLMs on custom datasets using techniques like LoRA and QLoRA. You'll build evaluation frameworks using both automated metrics and human evaluation rubrics. Equally important, you'll study AI safety and ethics: bias detection, red-teaming language models, content filtering, and responsible deployment practices. Understanding these issues isn't optional — it's what separates a competent AI engineer from a great one.

Phase 5: Capstone Project & Career Launch (Week 12)

Week 12 — Capstone Project & Demo Day

The final week is dedicated entirely to your capstone project — a portfolio-worthy AI application that you design, build, and present. Past capstone projects from our students have included:

  • An AI-powered legal document analyzer with RAG and citation extraction
  • A multi-agent system for automated market research
  • A real-time medical image classification tool deployed on AWS
  • A conversational AI tutor that adapts to student learning styles
  • An autonomous coding assistant built with LangGraph

On Demo Day, you'll present your project to an audience of hiring managers, industry mentors, and fellow students. It's a chance to showcase everything you've learned — and many students receive interview invitations directly from Demo Day.

Every project is designed to be portfolio-ready, meaning it includes clean code, documentation, a README, and deployment instructions. Here's a summary of the core projects:

  1. Exploratory Data Analysis Report — Clean, analyze, and visualize a real-world dataset.
  2. ML Classification Pipeline — Build and evaluate a complete machine learning pipeline with scikit-learn.
  3. Image Classifier with PyTorch — Train a CNN from scratch, achieve high accuracy, and analyze misclassifications.
  4. Sentiment Analysis System — Compare classical ML and deep learning approaches to NLP.
  5. AI Agent with Tool Use — Build an autonomous agent that reasons and acts using LangChain.
  6. RAG Knowledge-Base Chatbot — Create a retrieval-augmented chatbot deployed as a web service.

These projects aren't academic exercises. They're the kind of work AI engineers do every day at companies ranging from startups to Fortune 500 enterprises. When you walk into an interview, you won't be talking about hypotheticals — you'll be showing working code.
