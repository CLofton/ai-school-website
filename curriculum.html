12-Week Journey

The Curriculum

A rigorous, project-based curriculum designed by AI engineers who build production systems every day.

Week 1–2

Foundations of AI Engineering

  • Python for AI: Advanced Python patterns — async, generators, decorators, type hints
  • LLM Fundamentals: Transformer architecture, tokenization, attention mechanisms
  • Prompt Engineering: System prompts, few-shot learning, chain-of-thought, structured output
  • API Integration: OpenAI, Anthropic, and open-source model APIs
Project: Build a multi-model chatbot that routes queries to the best model for each task.
Week 3–4

RAG & Knowledge Systems

  • Vector Databases: Embeddings, similarity search, Pinecone, Weaviate, pgvector
  • Document Processing: PDF parsing, chunking strategies, metadata extraction
  • RAG Pipelines: Retrieval, reranking, context window management, citation
  • Evaluation: RAG metrics — faithfulness, relevance, answer correctness
Project: Build a RAG system over a 10,000-page legal document corpus with citation and source attribution.
Week 5–6

AI Agents & Tool Use

  • Agent Architecture: ReAct, function calling, planning loops, memory systems
  • Tool Integration: Building tools, API connectors, browser automation, code execution
  • Multi-Agent Systems: Agent coordination, delegation patterns, shared state
  • Safety & Guardrails: Input validation, output filtering, human-in-the-loop
Project: Build a 3-agent team (researcher, writer, reviewer) that produces publication-ready technical reports.
Week 7–8

Fine-Tuning & Model Optimization

  • Fine-Tuning: LoRA, QLoRA, PEFT, dataset preparation, training loops
  • Evaluation: Benchmarking, A/B testing, human evaluation protocols
  • Optimization: Quantization (GGUF, GPTQ), distillation, inference optimization
  • Open-Source Models: LLaMA, Mistral, Qwen — when to fine-tune vs. use APIs
Project: Fine-tune an open-source model on a custom dataset and deploy it with 10x lower latency than the base model.
Week 9–10

Production Systems & MLOps

  • Deployment: Docker, Kubernetes, serverless (Modal, Lambda), GPU cloud
  • Monitoring: LLM observability, cost tracking, latency dashboards, error budgets
  • Scaling: Load balancing, caching, rate limiting, batch processing
  • Security: Prompt injection defense, PII handling, audit logging, compliance
Project: Deploy a production AI service with monitoring, auto-scaling, and a full CI/CD pipeline.
Week 11–12

Capstone Project & Career Prep

  • Capstone: Design, build, and deploy a full AI agent system solving a real business problem
  • Demo Day: Present to hiring partners and Austin tech community
  • Portfolio: Polish GitHub, write technical blog posts, record demo videos
  • Career: Resume workshop, mock interviews, salary negotiation, networking
Capstone: Students choose from partner company challenges or propose their own. Past examples: AI customer support agent, automated financial analysis pipeline, multi-agent content creation system.
